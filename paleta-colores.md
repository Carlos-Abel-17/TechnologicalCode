# 🎨 TechnologicalCode - Color Palette (Futuristic Dark Theme)

---

## ⚫ Background (Fondos)
- Primary Background → #0A0A0A
- Secondary Background → #121212
- Tertiary Background → #1A1A1A

---

## 🧱 Surfaces (Cards, containers)
- Card Background → #1F1F1F
- Elevated Surface → #2A2A2A
- Border Subtle → #2F2F2F

---

## ⚪ Text (Tipografía)
- Primary Text → #FFFFFF
- Secondary Text → #CFCFCF
- Muted Text → #9A9A9A
- Disabled Text → #6B6B6B

---

## 🟣 Primary Accent (Elegir uno como principal)

### Opción 1 (Recomendada - Premium)
- Electric Purple → #7B61FF
- Purple Hover → #927BFF
- Purple Active → #5A45D6

### Opción 2 (Alternativa - Tech Startup)
- Neon Blue → #00D1FF
- Blue Hover → #33DAFF
- Blue Active → #009EC2

---

## 🌈 Gradients (Futuristic look)
- Main Gradient → #7B61FF → #00D1FF
- Dark Gradient → #121212 → #0A0A0A
- Glow Gradient → rgba(123,97,255,0.6) → rgba(0,209,255,0.6)

## ✨ Glow Effects (Importante para el estilo)
- Purple Glow → rgba(123, 97, 255, 0.6)
- Blue Glow → rgba(0, 209, 255, 0.6)
- Soft Glow → rgba(255, 255, 255, 0.1)

## 🔘 Buttons

### Primary Button
- Background → #7B61FF
- Text → #FFFFFF
- Hover → #927BFF
- Glow → rgba(123, 97, 255, 0.6)

### Secondary Button
- Background → transparent
- Border → #2F2F2F
- Text → #CFCFCF

---

## 🧩 UI States
- Success → #00FFB2
- Warning → #FFB020
- Error → #FF4D4D
- Info → #00D1FF

---

## 🧠 Usage Guidelines

- Usa fondo oscuro siempre (#0A0A0A)
- Máximo 1 color acento dominante (NO mezclar morado y azul sin control)
- Usa glow solo en elementos importantes (botones, hover, links)
- Mantén alto contraste para legibilidad
- Menos colores = más profesional

---

## 🚫 Avoid
- Rojo, verde o amarillo como colores principales
- Fondos claros
- Mezclar muchos acentos
- Colores saturados sin propósito

---

## 🚀 Pro Tip
El secreto del look futurista NO es el color...
es el uso de:
- sombras (glow)
- contrastes
- animaciones suaves
